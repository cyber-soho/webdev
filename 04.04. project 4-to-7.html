The Browser & DevTools

Understanding the rendering engine, pipeline, and debugging environment.

Weight: 1

Execution Steps

Pick two distinct engines (e.g., Chrome/Blink vs Firefox/Gecko). Open the same page in both.

  • Network Tab: Count HTTP requests, largest resources, load time.
  • Elements Panel: Find <body>, identify parent-child nodes. Edit text/style live.
  • Cross-Browser Check: Find 1 difference (font, form styling, etc.).

Use your project files for this:

  • Console: Cause a JS error, observe it, fix it.
  • Network (Disable Cache): Record requests, size, and DOMContentLoaded time.
  • Application: Store a value in localStorage and verify it.
