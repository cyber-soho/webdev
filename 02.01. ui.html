The 5 Core Principles

Mnemonic: C ¬∑ C ¬∑ F ¬∑ S ¬∑ A. These principles are the foundation of everything you build. Click each tab to learn the "One-Word Test" and see an example.

Clarity: "Is it obvious?"

The user should never have to guess. Every button and label needs an instant purpose.

Example: Which is clearer?

Consistency: "Is it predictable?"

Similar elements must look and behave the same. Builds trust and reduces the learning curve.

Feedback: "Does it respond?"

Every action produces a visible response. Click the button below to see feedback in action.

Action Registered!

Simplicity: "Is it necessary?"

Remove obstacles. Extra form fields and decorative clutter are enemies of conversion.
